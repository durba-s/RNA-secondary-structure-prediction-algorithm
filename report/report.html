Ribonucleic acid (RNA)

  • Ribonucleic acid (RNA) is a polymeric molecule essential in various biological roles in coding, decoding, regulation and expression of genes
  • The nucleic acid of RNA is ribose. The nucleotides that comprise RNA include adenine (A), guanine (G), cytosine (C) and uracil (U).
  • RNA is typically single-stranded and comes in various forms.
  • The single-stranded structure of RNA allows this molecule to fold back on itself and form various stable secondary structures as necessary.

Secondary structure

  • At a fundamental level, RNA secondary structure consists of nucleotides that are in one of two states, paired or unpaired, where pairing includes all base–base interactions. Thus understanding the secondary structure is essential for understanding the behavior of the RNA molecule.
  • Adenine (A) can pair with Uracil(U) and Guanine(G) can pair with Cytosine(C). It is to be noted that he set of base pairs forms a matching i.e each base can pair with atmost one base.
  • From a structural viewpoint, it is not feasible for the RNA molecule to bend too sharply, thus a pair of matching bases must be separated by atleast four intervening bases
  • In the following problem, we impose another condition on the secondary structure i.e the non-crossing condition between two base-pairings to approximate the spatial constraints on real RNA secondary structures.
  • Out of all the possible secondary structures of a RNA molecule, the secondary structure that is likely to exist under physiological conditions is the one with optimum free energy.
  • We assume that the free energy of a secondary structure is proportional to the number of base pairs that it contains. Thus in the problem to predict the secondary structure of a RNA molecule, we aim to predict the one with the maximum number of base pairings.

Problem Statement

Problem : Given an RNA sequence s, predict it’s optimal secondary structure.

Formulation :

  • An RNA sequence s is a string of bases; s = b1 b2…..bn where bi ∈ {A,U,C,G} for 1 ≤ i ≤ n
  • The optimal secondary structure predicted for s contains a set of base pairings B = { (i,j) } where 1 ≤ i < j ≤ n and the pairings satisfy the following rules:
    • [valid pairing] (S[i],S[j]) must be either one of (A,U) or (U,A) or (C,G) or (G,C)
    • [matching] Each i can pair with at most one j.
    • [at least 4 intervening bases in-between a pairing] 4 < j-i < n
    • [no crossing or no-knot] For every (i,j) and (k,l) ∈ B where i < k we cannot have i < k < j < l
  • We have to find the maximum number of base pairings satisfying the above constraints for the given RNA sequence s.

Algorithm

Recurrence relation: Let OPT(i,j) denote the maximum number of pairings possible between the indices i and j. This value will be the maximum of { OPT(i,j-1) (when s[i] does not pair with s[j]) and the maximum number of pairs obtained considering a base at index t (i ≤ t ≤ j-5) pairs with a base at index j (= 1 + OPT(i,t-1) + OPT(t+1,j)) } Also, because there must be atleast 4 intervening bases between a pairing OPT(i,j) = 0 whenever j-i < 5.

We have to find the optimal secondary structure for the entire sequence, thus the maximum number of pairings is OPT(1,n) which can be obtained by solving the above recurrence.

To obtain the matching pairs we recursively trace back the corresponding (i,j) for which the max value was obtained while calculating the OPT(i,j) for the subproblems.
