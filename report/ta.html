Theoretical Analysis

Computing the maximum number of pairings for a sequence: The time complexity is O(n3) (n = length of the sequence; Since we have to iterate over diff (varies from 5 to n-1) and i (varies from 1 to n-diff); and compute the recurrence over t (varies from i+1 to j-5 where j=i+diff) for every pair (diff,i) )

Backtracking Step: The time complexity is O(k) (k is the maximum number of pairings obtained)

Therefore the overall time complexity is O(n3). (O(n3)+O(k) = O(n3)+ O(n) [since k is the maximum number of pairings obtained, so k < n ] = O(n3) )
